This program is a Garage Management System written in C++ that helps track car services for customers. It allows a manager to register customers and their vehicles, automatically assigning unique Car IDs and Spare Part IDs.
Customers can use the system to request washing or maintenance services.
A key feature is the security verification step: after a car is maintained, the customer must enter the correct Spare Part ID to ensure no parts were swapped incorrectly before the car's status is updated to "Running."
